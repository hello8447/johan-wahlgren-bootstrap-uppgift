Bootstrapp-begrepp

Vad är ett Element?

Ett element är information som är innanför en tag

Allt som ni kan se nu är ett element innanför en paragraph tag

Vad är HTML?

HTML står för hyper text markup language och är de största byggstenarna till en hemsida. HTML är den enklaste och äldsta programmeringspårket

Allt jag skriver nu är skriven i HTML

Vad är Id?

ID används när man specefikt vill endast vill ändra en specefik egenskap på ett element som endast ska användas en eller två gånger som t.ex en del av texten i en container ska vara röd

Man kan endast ha en Id per fil

Vad menas med class?

Class används för att ge egenskaper till ett element som t.ex att en container ska vara röd eller ifall man klickar på en länk kommer den komma till en annan hemsida.

Vad betyder meta?

I en meta-tagg brukar man skriva in viktig information om sin hemsida. Meta taggen ligger alltid i head elementet. Det kan vara information Som t.ex:

  • vem som ligger bakom hemsidan
  • nyckelord
  • sponsorer till sidan
  • hemsidans ämne
  • källor till sidan

Vad menas med Responsive?

Responsive betyder att hemsidan ska vara smidigt att använda och se bra ut på båda mobilen och på datorn Exempel på detta kan vara

Vad betyder HTTP?

HTTP står för HyperText Transfer Protocol,HTTP styr hur data hämtas och skickas mellan hemsidor och servrar runt hela världen

Vad betyder HTTPS?

HTTPS står för Secure HyperText Transfer Protocol och är en säkrare väg till HTTP datan. När man är på en webbsida vill man alltid att hemisdan ska vara HTTPS

Vad menas med compressed data?

Compressed data är minne eller data som har minskats vanligtvis för att man har slut på minne i en dator eller för att man vill ladda enr en fil som har för mycket data

Vad betyder CDN?

CDN står för Content Delivery Network och jobbet att minska lagg på en server genom minska distansen mellan användaren och servern

Vad är ett local bootstrap?

Local bootstrap är ett biblotek av flera tusen CSS koder som man kan kopiera ifrån och använda till sitt egna bruk.

Local bootstrap används när man vill göra en hemsida mobilanpassad och responsive.

Vad betyder JQueary?

JQueary är precis som local bootstrap ett bibloiotek fast med javascrpit som kan förenkla din CSS och HTML

Vad menas med Flexbox?

Hela den här hemisdna är gjord med hjälp av Flexbox

Flexbox är byggd på rektanglar som kallas flera containers som man använder för att ge en snyggare struktur och finess till din hemsidan.

Vad menas med Grid?

Grid bestämmer designen och "layoyten" på din hemsida.

Med grind kan man bestämma saker som:

  1. Vart titeln ska placeras
  2. Hur många conatainers som ska få plats lodrätt och vågrätt
  3. Vart footern ska vara
  4. Bara allmänt designa sidan

Vad är code-snippets?

Code snippets är ett begrepp inom programmeirng som beskriver en liten grupp av kod

Som t.ex koden i denna containern är code-snippets
